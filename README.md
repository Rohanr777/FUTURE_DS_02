# 📊 Customer Churn Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing customer churn behavior using Excel. The objective is to identify key factors influencing customer retention and provide actionable insights to reduce customer loss.

---

## 📁 Dataset Features

* CustomerID, Age, Gender
* Tenure, Usage Frequency
* Support Calls, Payment Delay
* Subscription Type, Contract Length
* Total Spend, Last Interaction
* Churn (Target Variable)

---

## 📊 Key KPIs

* Total Customers: **64,374**
* Churn Rate: **~32%**
* Retention Rate: **~68%**

## 🔍 Key Insights

* Customers with **high payment delays** show the highest churn rates
* **Low activity users** are more likely to leave the service
* **Basic subscription plans** have higher churn compared to premium plans
* **Adult age group** contributes the most to churn volume
* Customers with **frequent support calls** indicate dissatisfaction and higher churn risk

---

## 📈 Analysis Performed

* Data cleaning and preprocessing in Excel
* Churn rate and retention calculation
* Customer segmentation (Age, Subscription, Activity)
* KPI creation and dashboard design
